# NexusArena - Esports Tournament Management Database

## Overview
NexusArena is a fully normalized MySQL database designed to manage professional esports tournaments, teams, players, sponsorships, and prize distributions.

## Features
- Third Normal Form (3NF)
- Player and Team Management
- Tournament Scheduling
- Match Tracking
- Sponsorship Management
- Prize Distribution
- SQL Triggers
- Analytical Views

## Technologies Used
- MySQL
- SQL
- JavaFX
- Draw.io
