\# NexusArena - Esports Tournament Management Database



\## Overview



NexusArena is a fully normalized relational database system designed to manage professional esports tournaments, teams, players, sponsorships, and prize distributions.



\## Features



\- Third Normal Form (3NF)

\- 10 relational tables

\- 3 analytical views

\- 2 triggers

\- JavaFX GUI integration



\## Technologies



\- MySQL 8.x

\- SQL

\- JavaFX

\- Draw.io
